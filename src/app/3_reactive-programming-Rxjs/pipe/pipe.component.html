Pipe est une méthode sur Observable utilisée pour la composition des opérateurs.
pipe a été introduit dans RxJS dans la v5.5 .

Sans la méthode pipe

of(1,2,3,4,5)
.map(value => value *2 )
.filter(value => value > 4)

Avec la méthode pipe

of(1,2,3,4,5).pipe(
map(value => value *2 ),
filter(value => value > 4)
);

Même résultat, même concept (opérateurs de composition), syntaxe différente.

pipe offre beaucoup des avantages ,et voila quelques avantages :

* Il nettoie Observable.prototype en supprimant des opérateurs

* Cela rend la bibliothèque RxJS plus puissant ...,

* Cela facilite l’écriture et l’utilisation d’opérateurs tiers

(puisque vous n’aurez plus à vous soucier de patcher Observable.prototype).
