Subject

Un Subject est un type spécial d'Observable qui permet de multidiffuser des valeurs à de nombreux observateurs. Les Subjects sont comme EventEmitters.

Exemple code

{{"const sub = new Subject();"}}
{{"sub.next(1);"}}
{{"sub.subscribe(res => console.log('Result sub 1;', res));"}}
{{"sub.next(2);"}}
{{"sub.subscribe(res => console.log('Result sub 2:', res));"}}
{{"sub.next(3);"}}

Result sub 1: 2
Result sub 1: 3
Result sub 2: 3
