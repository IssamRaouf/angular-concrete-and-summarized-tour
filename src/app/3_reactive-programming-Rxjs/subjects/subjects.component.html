Description :

Un sujet est un type spécial d'Observable qui partage un seul chemin d'exécution entre les observateurs.
Vous pouvez penser à cela comme un seul orateur parlant dans un microphone dans une pièce pleine de monde.
Leur message (le sujet) est transmis à de nombreuses personnes (multidiffusion) (les observateurs) à la fois.
C'est la base de la multidiffusion. Les observables typiques seraient comparables à une conversation en tête-à-tête.
