BehaviorSubject

Une variante de Subject qui nécessite une valeur initiale et émet sa valeur actuelle chaque fois qu'il est abonné.

Exemple code

{{"const source = new BehaviorSubject(123);"}}
{{"source.subscribe(res => console.log('Result subs 1:', res));"}}
{{"source.subscribe(res => console.log('Result subs 2:', res));"}}
{{"source.next(333);"}}
{{"source.complete();"}}

Result subs 1: 123
Result subs 2: 123
Result subs 1: 333
Result subs 2: 333
