AsyncSubject

Une variante de Subject qui n'émet une valeur qu'à la fin. Il émettra sa dernière valeur à tous ses observateurs une fois terminé.

Exemple code

{{"const source = new AsyncSubject();"}}
{{"source.subscribe(res => console.log('Result subs 1', res));"}}
{{"source.next(333);"}}
{{"source.subscribe(res => console.log('Result subs 2', res));"}}
{{"source.next(444);"}}
{{"source.complete();"}}

Result subs 1 : 444
Result subs 2 : 444
