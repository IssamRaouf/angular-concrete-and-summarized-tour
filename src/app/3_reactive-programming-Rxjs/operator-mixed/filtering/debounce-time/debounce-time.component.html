DebounceTime

Émet une valeur de la source Observable uniquement après une période donnée sans autre émission source.

Exemple code

{{"const source = interval(1000).pipe(map(val => `Time : ${val + 1} second`), take(10));"}}
{{" const result = source.pipe(debounceTime(2000));"}}
{{"result.subscribe(res => console.log('Result: ', res));"}}
