SkipWhile

Renvoie un observable qui ignore tous les éléments émis par la source Observable tant qu'une condition spécifiée reste vraie, mais émet tous les autres éléments source dès que la condition devient fausse.

Exemple code

{{" const source = interval(1000).pipe(take(10));"}}
{{"const result = source.pipe(skipWhile(item => item < 8));"}}
{{"result.subscribe(res => console.log('Result : ', res));"}}

Result : 8
Result : 9
