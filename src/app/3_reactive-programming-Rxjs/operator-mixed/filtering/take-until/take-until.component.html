TakeUntil

Émet les valeurs émises par la source Observable jusqu'à ce qu'un notifiant Observable émette une valeur.

Exemple code

{{"const source = interval(1000).pipe(takeUntil(timer(5000)));"}}
{{"source.subscribe(res => console.log('Result : ', res));"}}

Result : 0
Result : 1
Result : 2
Result : 3
