TakeLast

N'émet que les dernières valeurs de comptage émises par la source Observable.

Exemple code

{{"const source = of('Issam', 'Soufiane', 'Raouf', 'Ramouda');"}}
{{"const result = source.pipe(takeLast(2));"}}
{{"result.subscribe(res => console.log('Result : ', res));"}}

Result : Raouf
Result : Ramouda
