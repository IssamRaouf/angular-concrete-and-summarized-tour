Sample

Émet la dernière valeur émise par la source Observable à chaque fois qu'un autre Observable, le notifiant, émet

Exemple code

{{"const source = interval(1000).pipe(take(10));"}}
{{"const result = source.pipe(sample(interval(2000)));"}}
{{"result.subscribe(res => console.log('Result :', res));"}}

Result : 1
Result : 3
Result : 5
Result : 7
