Filter

Filtrer les éléments émis par la source Observable en n'émettant que ceux qui satisfont un prédicat spécifié. .

Exemple code

{{" const source = of(1, 2, 3, 4, 5, 6);"}}
{{" const result = source.pipe(filter(num => num % 2 === 0));"}}
{{" result.subscribe(res => console.log('Result : ', res));"}}

Result : 2
Result : 4
Result : 6
