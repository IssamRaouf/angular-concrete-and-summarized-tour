DistinctUntilKeyChanged

Renvoie un observable qui émet tous les éléments émis par l'observateur source qui sont distincts par comparaison de l'élément précédent, à l'aide d'une propriété accessible à l'aide de la clé fournie pour vérifier si les deux éléments sont distincts.

Exemple code

{{"const source = of({age: 4, name: 'Foo'}, {age: 7, name: 'Bar'}, {age: 5, name: 'Foo'}, {age: 6, name: 'Foo'});"}}
{{"const result = source.pipe(distinctUntilKeyChanged('name'));"}}
{{"result.subscribe(res => console.log('Result : ', res));"}}

Result : {{"{age: 4, name: 'Foo'}"}}
Result : {{"{age: 7, name: 'Bar'}"}}
Result : {{"{age: 5, name: 'Foo'}"}}
