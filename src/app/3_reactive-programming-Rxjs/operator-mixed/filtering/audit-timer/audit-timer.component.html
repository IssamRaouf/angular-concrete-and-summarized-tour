AuditTimer

Ignore les valeurs source pour la durée, en millisecondes, puis émet la valeur la plus récente de l'observable source, puis répète ce processus.

Exemple code

{{"const source = fromEvent(document, 'click');"}}
{{"const result = source.pipe(auditTime(1000));"}}
{{"result.subscribe(res => console.log('Result : ', res));"}}
