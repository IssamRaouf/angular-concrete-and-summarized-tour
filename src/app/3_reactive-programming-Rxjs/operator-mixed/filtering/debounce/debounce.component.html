Debounce

N'émet une valeur de la source Observable qu'après un laps de temps déterminé par un autre observable, sans émission d'une autre source.

Exemple code

{{"const source = of('HELLO', 'WORLD', 'GOODBYE', 'WORLD');"}}
{{"const result = source.pipe(debounce(() => timer(1000)));"}}
{{"result.subscribe(res => console.log('RESULT :', res));"}}
