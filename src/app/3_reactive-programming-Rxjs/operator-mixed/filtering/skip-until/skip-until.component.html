SkipUntil

Renvoie un observable qui ignore les éléments émis par la source Observable jusqu'à ce qu'un deuxième Observable émette un élément.

Exemple code

{{" const source = interval(1000).pipe(take(10));"}}
{{" const result = source.pipe(skipUntil(timer(6000)));"}}
{{"result.subscribe(res => console.log('Result : ', res));"}}

Result : 5
Result : 6
Result : 7
Result : 8
Result : 9
