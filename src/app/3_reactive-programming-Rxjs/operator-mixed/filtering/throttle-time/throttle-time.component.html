ThrottleTime

Émet une valeur à partir de l'observable source, puis ignore les valeurs source suivantes pour la durée, en millisecondes, puis répète ce processus.

Exemple code

{{"const source = interval(1000).pipe(take(10));"}}
{{"const result = source.pipe(throttleTime(2000));"}}
{{"result.subscribe(res => console.log('Result :', res));"}}

Result : 0
Result : 1
Result : 3
Result : 6
Result : 9
