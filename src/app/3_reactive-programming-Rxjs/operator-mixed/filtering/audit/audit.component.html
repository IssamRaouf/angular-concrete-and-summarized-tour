Audit

Ignore les valeurs source pour une durée déterminée par un autre observable, puis émet la valeur la plus récente de l'observateur source, puis répète ce processus.

Exemple code

{{"const source = fromEvent(document, 'click');"}}
{{"const result = source.pipe(audit(ev => interval(3000)));"}}
{{"result.subscribe(x => {"}}
{{"console.log(x);"}}
{{"this.resultIn = x;"}}
{{"});"}}
{{"result.subscribe(x => console.log(x));"}}
