Repeat

Renvoie un observable qui se réabonnera au flux source lorsque le flux source sera terminé, au plus des fois

Exemple code

{{"const source = of('Issam', 'Raouf');"}}
{{"const result = source.pipe(repeat(2));"}}
{{"result.subscribe(res => console.log('Result :', res));"}}

Result : Issam
Result : Raouf
Result : Issam
Result : Raouf
