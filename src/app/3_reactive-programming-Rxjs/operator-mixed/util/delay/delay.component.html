Delay

Retarde l’émission d’objets de la source Observable selon un délai donné ou jusqu’à une date donnée.

Exemple code

{{"const source = of(1);"}}
{{"const result = merge("}}
{{"source.pipe(mapTo('Hello'), delay(100)),"}}
{{"source.pipe(mapTo(' World'), delay(200)),"}}
{{"source.pipe(mapTo(' Goodbye'), delay(300)),"}}
{{"source.pipe(mapTo(' World'), delay(400)),"}}
{{");"}}
{{"result.subscribe(res => console.log('Result', res));"}}

Result : Hello
Result : World
Result : Goodbye
Result : World
