DelayWhen

Retarde l’émission d’objets de la source Observable d’une durée donnée, déterminée par les émissions d’un autre Observable

Exemple code

{{"const source = interval(1000).pipe(take(5));"}}
{{"const result = source.pipe(delayWhen(() => timer(5000)));"}}
{{"result.subscribe(res => console.log('Result :',res));"}}

Result : 0
Result : 1
Result : 2
Result : 3
Result : 4
