SwitchMap

Projette chaque valeur source sur un observable qui est fusionnée dans la sortie Observable, en émettant des valeurs uniquement à partir du dernier observable projeté.

Exemple code

{{"const source = of(26, 44, 66);"}}
{{"const result = source.pipe(switchMap(age => of({age)));"}}
{{"result.subscribe(res => console.log('Result :', res));"}}

Result : {{"{age: 26}"}}
Result : {{"{age: 44}"}}
Result : {{"{age: 66}"}}
