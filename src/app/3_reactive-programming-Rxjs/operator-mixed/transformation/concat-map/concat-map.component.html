Buffer

Projette chaque valeur source dans un observable qui est fusionné dans l'observable de sortie, de manière sérialisée, en attendant que chacune d'elles se termine avant de fusionner la suivante.

Exemple code

{{"const source = of(2000, 4000, 6000);"}}
{{"const result = source.pipe(concatMap(val => of(`Delay of : ${val} ms`).pipe(delay(val))));"}}
{{"result.subscribe(res => console.log('Result :', res));"}}
