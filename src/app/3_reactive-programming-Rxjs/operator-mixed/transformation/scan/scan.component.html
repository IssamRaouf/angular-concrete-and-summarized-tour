Scan

Applique une fonction d'accumulation sur l'observable source et renvoie chaque résultat intermédiaire avec une valeur de départ facultative.

Exemple code

{{" const source = of(3, 7, 9);"}}
{{"const result = source.pipe(scan((accu, curVal) => accu + curVal, 0));"}}
{{"result.subscribe(res => console.log('Result :', res));"}}

Result : 3
Result : 10
Result : 19
