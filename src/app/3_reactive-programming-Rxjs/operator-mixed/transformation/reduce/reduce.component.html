Reduce

Applique une fonction d'accumulateur sur l'observable source et renvoie le résultat accumulé à la fin de la source, en fonction d'une valeur de départ facultative.

Exemple code

{{" const source = of(3, 7, 9);"}}
{{"const result = source.pipe(reduce((accu, curVal) => accu + curVal, 0));"}}
{{"result.subscribe(res => console.log('Result :', res));"}}

Result : 19
