MapTo

Émet la valeur constante donnée sur la sortie Observable à chaque fois que la source Observable émet une valeur.

Exemple code

{{" const source = fromEvent(document, 'click');"}}
{{" const result = source.pipe(mapTo('click In'));"}}
{{" result.subscribe(res => console.log('Result ', res));"}}

chaque clique
Result : click In
