Buffer

Met en mémoire tampon les valeurs observables source jusqu'à ce que fermetureNotifier émette.

Exemple code

{{" const fermetureNotifier = fromEvent(document, 'click');"}}
{{" const source = interval(1000);"}}
{{" const result = source.pipe(buffer(fermetureNotifier));"}}
{{"result.subscribe(res => console.log('Result :', res));"}}

exemple avec trois clique sur le document Result : (2) [0, 1]
Result : (3) [2, 3, 4]
Result : (4) [5, 6, 7, 8]
