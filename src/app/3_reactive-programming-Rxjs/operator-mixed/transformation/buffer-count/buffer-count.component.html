BufferCount

Met en mémoire tampon les valeurs observables source ( Recueillez les valeurs de sortie..) jusqu'à ce que fermetureNotifier émette.

Exemple code

{{"const source = interval(1000).pipe(take(10));"}}
{{"const result = source.pipe(bufferCount(2));"}}
{{"result.subscribe(res => console.log('Result : ', res));"}}

Result :  [0, 1]
Result :  [2, 3]
Result :  [4, 5]
Result :  [6, 7]
Result :  [8, 9]
