ToArray

Collecte toutes les émissions sources et les émet sous forme de tableau lorsque la source est terminée.

Exemple code

{{"const source = of(2, 4, 6, 8, 10);"}}
{{"const result = source.pipe(toArray());"}}
{{"result.subscribe(res => console.log('Result :', res));"}}

Result : [2, 4, 6, 8, 10]
