BufferTime

Met en mémoire tampon les valeurs observables source pour une période donnée.

Exemple code

{{"const source = interval(500).pipe(take(10));"}}
{{"const result = source.pipe(bufferTime(2000));"}}
{{"result.subscribe(res => console.log('Result : ', res));"}}

Result :  [0, 1, 2]
Result :  [3, 4, 5, 6]
Result :  [7, 8, 9]
